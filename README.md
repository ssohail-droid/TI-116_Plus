##Required Components & Tools:
- TI-84 Plus
- ESP32
- Bidirectional 5V-to-3.3V level shifter
- Wires
- Soldering iron
- Heat shrink tubing
- Electrical tape
- Torx T6 screwdriver (for the outer screws)  
- Phillips screwdriver (for the backup battery)
